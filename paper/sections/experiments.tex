\section{Experimental Setup}
\label{sec:setup}

\subsection{Data}
\label{sec:data}

\paragraph{California (in-distribution).} We use the Northern California Seismic Network (NCSN) via NCEDC as an in-distribution baseline, as PhaseNet was trained on NCSN data. We select M3.5--5.0 events for 2020--2023: the lower bound ensures clear arrivals above noise, the upper bound avoids instrument saturation, and the time window matches the ISC Reviewed Bulletin coverage used for Kazakhstan. Three-component broadband waveforms are downloaded from the 8 closest BK network stations within $2^\circ$ of each event, as PhaseNet requires all three components to discriminate P- and S-wave arrivals. Ground truth picks are taken from the NCEDC catalog (273 P-wave and 12 S-wave arrivals across 325 waveforms), since this is the same source as the waveforms, ensuring the picks and recordings refer to the same events.

\paragraph{Kazakhstan (out-of-distribution).} We deploy PhaseNet to the KZ network via IRIS FDSN as our out-of-distribution test case, since Kazakhstan is seismically active but lies entirely outside PhaseNet's training distribution. Starting from 29 three-component broadband (BH*) stations, we apply a filtering pipeline: remove experimental and temporary deployments (which lack the long-term stability needed for a four-year study), verify waveform availability at four test times (to confirm data is actually retrievable before committing to a station), confirm instrument stability with no mid-study sensor changes (so any performance shifts can be attributed to seismic regime rather than hardware), and require ISC pick coverage (stations without ground truth picks cannot contribute to evaluation). This yields 11 stations spanning six broadband sensor types (STS-6, STS-2, CMG-3TB, KS-54000, CMG-3ESP, Trillium 120), all sampling at 40\,Hz. The study window is 2020--2023, chosen to match the upper limit of the ISC Reviewed Bulletin's coverage and to ensure all retained stations are on stable hardware throughout. Ground truth is taken from the ISC Reviewed Bulletin \citep{isc2023}, which provides human-analyst-verified picks integrating data from regional agencies, giving more accurate hypocentral solutions for Central Asian events than global catalogs alone. This yields 4{,}674 verified picks across the 11 stations. PhaseNet produces 190 picks evaluated against 202 P-wave and 74 S-wave ISC arrivals.

\paragraph{Regime data.} We query the ISC earthquake catalog for Central Asia (41--55$^\circ$N, 55--85$^\circ$E), obtaining 6{,}763 M$\geq$3.0 events for weekly count observations and 12 M$\geq$4.5 events as potential regime triggers. Only 4 of these fall within 200\,km of any station, meaning 7 of 11 stations experience no regime transitions during the study period. Full data pipeline details are provided in \cref{app:data}. \Cref{tab:data} summarises both datasets.

\subsection{Evaluation Protocol}
\label{sec:eval_protocol}

PhaseNet is run with original weights (trained on Northern California) using SeisBench \citep{woollam2022seisbench}. Waveforms are 120\,s windows starting 10\,s before the event origin time, preprocessed with detrending, tapering, bandpass filtering (1--45\,Hz), and normalisation. Picks are extracted with detection thresholds of 0.3 for both P and S phases.

A PhaseNet pick is considered correct if it falls within 0.5\,s of a catalog arrival of the same phase type. This tolerance is consistent with the original PhaseNet evaluation \citep{zhu2019phasenet}. We report detection rate (recall), precision, pick residuals (mean and standard deviation), Expected Calibration Error (ECE), and AUROC for correct/incorrect discrimination. Both calibration and discrimination metrics are computed for P-wave picks separately because S-wave ground truth is limited (12 picks in California vs.\ 74 in Kazakhstan), making cross-region S-wave comparison unreliable.

\subsection{Baselines and Comparisons}
\label{sec:baselines}

We compare three approaches at matched deferral rates:
\begin{itemize}
    \item \textbf{No deferral:} accept all PhaseNet picks, establishing the lower bound (0.432 system accuracy in Kazakhstan).
    \item \textbf{Confidence-only deferral:} defer picks with the lowest PhaseNet confidence scores, the standard approach using model self-assessment.
    \item \textbf{Combined score (proposed):} defer picks with the lowest regime-aware deferral scores (\cref{eq:deferral_score}), incorporating environmental context.
\end{itemize}
All three are compared at a fixed 25\% deferral rate (48 of 190 picks) to isolate the effect of the regime signal.
